全新安装：
1.安装 umi 依赖: cnpm install umi --save-dev
2.添加 umi-plugin-react 插件: cnpm install umi-plugin-react --save-dev


更新安装：
1.安装 package.json文件中指定版本的依赖包: yarn（或 cnpm install）
2.启动项目服务：
方式一：npm run start
方式二：cnpm run start
方式三：yarn run start
方式四：umi dev
方式五：pm2 start npm -- run start


访问页面：
1.访问：http://localhost:8001/
